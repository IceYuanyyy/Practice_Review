# 题库转换工具 - 兼容格式标准

> 本文档列出了 `QuestionConverter.vue` 支持的所有转换格式。

---

## 一、文件格式要求

| 项目 | 支持格式 |
|------|---------|
| 文件类型 | `.txt` 文本文件 |
| 编码格式 | `UTF-8` / `GBK` |

---

## 二、题号格式

以下格式均可识别：

```
1.题目内容
1、题目内容
1 题目内容
1题目内容
```

**示例：**
```
1.计算机网络分为哪几层？
2、OSI模型是什么？
3 TCP和UDP的区别是什么？
```

---

## 三、选项格式

支持 **A-F** 共 6 个选项，以下分隔符均可识别：

| 格式 | 示例 |
|------|------|
| 英文句点 | `A.选项内容` |
| 中文顿号 | `A、选项内容` |
| 英文冒号 | `A:选项内容` |
| 中文冒号 | `A：选项内容` |
| 空格 | `A 选项内容` |

**示例：**
```
1.以下哪个是传输层协议？
A.HTTP
B、TCP
C：UDP
D FTP
E:SMTP
F、DNS
```

---

## 四、答案区格式

> [!IMPORTANT]
> 答案区位于文件末尾，格式为：`起始题号-结束题号 答案1 答案2 答案3 ...`

### 选择题答案

| 支持格式 | 说明 |
|---------|------|
| `A` `B` `C` `D` `E` `F` | 单选答案 |
| `AB` `ACD` `BCE` | 多选答案（连写） |

### 判断题答案

| 格式 | 等效值 |
|------|-------|
| `T` / `True` / `对` / `√` / `正确` | ✓ 正确 |
| `F` / `False` / `错` / `×` / `错误` | ✗ 错误 |

**示例：**
```
1-5 A B C D A
6-10 B C A D C
11-15 对 错 √ × T
16-20 True False 正确 错误 F
```

---

## 五、判断题题内答案格式

判断题可在题目后直接标注答案，以下格式均可识别：

```
答案：正确
答案：错误
答案：对
答案：错
答案：√
答案：×
答案：T
答案：F
答案：True
答案：False
```

> [!NOTE]
> 系统会自动将所有格式标准化为 `正确` 或 `错误`

**示例：**
```
1.TCP是面向连接的协议
答案：正确

2.UDP提供可靠传输
答案：错
```

---

## 六、解析格式

以下格式均可识别：

```
解析：解析内容
答案解析：解析内容
```

**示例：**
```
1.HTTP默认端口号是？
A.21
B.80
C.443
D.8080
解析：HTTP协议默认使用80端口
```

---

## 七、图片URL格式

以下格式均可识别：

```
图片：URL地址
Image：URL地址
```

**示例：**
```
1.如图所示，该网络拓扑属于什么类型？
图片：https://example.com/network.png
A.星型
B.环型
C.总线型
D.网状
```

```txt
B.FTP
C.TCP
D.SMTP
E.UDP

1-3 C A ACE
```
